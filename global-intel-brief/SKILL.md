# Global Intelligence Brief Skill

## 描述
这是一个自动化的全球情报简报生成工具，可以从多个国际新闻源获取信息，进行聚类分析，并生成结构化的全球情报简报。该技能专注于地缘政治、区域冲突和宏观经济领域的信息收集与分析。

## 功能
- 自动抓取 BBC、The Guardian、Al Jazeera、South China Morning Post 等主流国际新闻源
- 智能聚类和去重，避免信息冗余
- 综合多信源信息，形成完整叙述
- 优先引用带具体数据、地理位置、政策条款的硬新闻
- 按照预设模板生成结构化情报简报

## 输入
- 新闻源URL列表
- 分析领域设定（地缘政治、区域冲突、宏观经济等）
- 简报生成时间

## 输出
- 结构化情报简报，包含：
  - 各板块核心研判
  - 详细子话题分析
  - 信源整合标注
  - 风险提示与机会洞察

## 依赖
- DeepSeek API (用于内容分析)
- Telegraph API (用于发布)
- Python 3 及相关库
- curl 和 jq 工具

## 使用场景
- 日常全球情报监测
- 地缘政治风险评估
- 宏观经济趋势分析
- 多信源信息验证

## 配置
- API密钥配置
- 新闻源URL列表
- 简报发布时间设置
- 目标发布平台配置

## 注意事项
- 需要稳定的网络连接
- API调用可能需要一定时间
- 需要定期更新新闻源URL
