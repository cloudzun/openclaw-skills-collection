# Simple Weather Forecast Skill

一个简化的天气预报查询技能，提供指定地点的基本天气信息，避免了长时间等待详细预报的问题。

## 功能特性

- 查询当前天气状况（温度、体感温度、湿度、风速等）
- 获取当天天气预报
- 支持中英文显示
- 快速响应，避免长时间等待
- 支持全球任意地点查询

## 使用方法

### 基本用法
```bash
# 使用默认位置（上海闵行区）
./weather_simple.sh

# 指定邮编或城市名
./weather_simple.sh 201108
./weather_simple.sh Beijing
./weather_simple.sh "New York"
```

### 参数说明

- `location`: 要查询的位置，可以是邮编、城市名或地区名，默认为 `201108`（上海闵行）

## 依赖

- `curl` 或 `wget` - 用于请求天气数据
- `wttr.in` API - 天气数据来源

## 返回内容

- 当前温度和体感温度
- 湿度、风速信息
- 天气图标和简要描述
- 当天天气预报

## 示例

```bash
# 上海闵行区天气
./weather_simple.sh 201108

# 北京天气
./weather_simple.sh Beijing
```
