# 每日科技巨头战略简报技能

## 描述
这是一个经过验证的、每日运行的科技简报工具，从多个权威科技新闻源获取信息，生成专业的科技简报。该技能使用经过验证的脚本每天早上6:15自动运行。

## 功能
- 自动抓取 The Verge、TechCrunch、SCMP、The Guardian、BBC 等科技新闻源
- 智能分析AI大模型、芯片硬件、造车新势力、互联网巨头等领域
- 按照预设模板生成结构化科技简报
- 通过Telegraph发布完整版本

## 输入
- 科技新闻源URL列表
- 分析领域设定（AI、芯片、互联网等）
- 简报生成时间

## 输出
- 结构化科技简报，包含：
  - 市场情绪概述
  - 3-5个最有价值的板块分析
  - 分析师关注（风险/机会）

## 依赖
- DeepSeek API (用于内容生成)
- Telegraph API (用于发布)
- Python 3 及相关库
- curl 和 jq 工具

## 使用场景
- 日常科技动态监测
- AI行业趋势分析
- 芯片行业动态追踪
- 互联网巨头战略观察

## 配置
- API密钥配置
- 新闻源URL列表
- 简报发布时间设置（默认每天6:15）

## 注意事项
- 需要稳定的网络连接
- API调用可能需要一定时间
- 需要定期检查API密钥有效性
