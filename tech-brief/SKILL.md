# Tech Brief Skill

## 描述
这是一个自动化的科技简报生成工具，可以从多个科技新闻源获取信息，进行分析，并生成结构化的科技简报。该技能专注于AI大模型、芯片/硬件、造车新势力、互联网巨头等领域的信息收集与分析。

## 功能
- 自动抓取 The Verge、TechCrunch、SCMP、The Guardian、BBC 等科技新闻源
- 智能分析AI、芯片、互联网等行业动态
- 按照预设模板生成结构化科技简报
- 支持多种输出格式（包含/不包含 Telegraph 发布）

## 输入
- 科技新闻源URL列表
- 分析领域设定（AI、芯片、互联网等）
- 简报生成时间

## 输出
- 结构化科技简报，包含：
  - 市场情绪概述
  - AI动态板块
  - 芯片行业板块
  - 其他科技领域板块
  - 分析师关注（风险/机会）

## 依赖
- DeepSeek API (用于内容生成)
- Telegraph API (用于发布，可选)
- Python 3 及相关库
- curl 和 jq 工具

## 使用场景
- 日常科技动态监测
- AI行业趋势分析
- 芯片行业动态追踪
- 互联网巨头战略观察

## 配置
- API密钥配置
- 新闻源URL列表
- 简报发布时间设置
- 目标发布平台配置（可选）

## 版本
- main.sh: 主要版本，包含 Telegraph 发布功能
- no-telegraph.sh: 不包含 Telegraph 发布功能
- minimal.sh: 最简化版本
- debug.sh: 调试版本
- debug-full.sh: 完整调试版本
